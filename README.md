# INFUSION - Cafetería para Programadores

Infusion es una landing page moderna desarrollada para una cafetería temática orientada a desarrolladores, estudiantes de tecnología y entusiastas del software.

## Tecnologías Utilizadas
- HTML5 Semántico
- CSS3 (Variables, Grid, Flexbox, Animaciones)
- JavaScript Vanilla (DOM, Eventos, IntersectionObserver)
- Metodología Mobile First

## Estructura
- `/css`: Archivos de estilos separados (variables, estilos base y responsive).
- `/js`: Lógica de interacción (menú, validaciones, contadores, scroll animado).
- `/assets`: Recursos visuales.

## Cómo ejecutar el proyecto
1. Clona o descarga el repositorio.
2. Abre el archivo `index.html` en tu navegador web preferido. No requiere servidor local, aunque se recomienda usar Live Server en VS Code para una mejor experiencia de desarrollo.
